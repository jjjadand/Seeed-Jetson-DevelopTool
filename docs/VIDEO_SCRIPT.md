# Seeed Jetson Develop Tool — 产品视频拍摄脚本

**视频定位**：产品功能演示 / 官网 & 社媒推广  
**目标时长**：3 分 30 秒 ～ 4 分钟  
**目标受众**：Jetson 开发者、嵌入式 AI 工程师、Seeed 产品用户  
**拍摄形式**：屏幕录制 + 实物镜头 + 旁白字幕

## 拍摄注意事项

**屏幕录制**
- 分辨率建议 1920×1080，录制帧率 60fps
- 鼠标移动要慢而稳，点击前停顿 0.5 秒
- 终端字体放大到 16pt 以上，确保字幕清晰
- 使用深色终端主题（与 GUI 风格统一）

**实物镜头**
- 开场 reComputer 设备特写：侧光打亮，背景简洁（黑色或深灰）
- USB 线连接主机的画面：展示刷机物理连接

**剪辑节奏**
- 安装/下载等耗时操作使用快进（2x ～ 4x）
- 关键操作（点击、选择、结果出现）保持正常速度
- 每个模块切换时加 0.3 秒黑场过渡

**配乐**
- 背景音乐：轻快科技感纯音乐，音量压低（旁白优先）
- 操作成功音效：短促的「叮」声

**字幕**
- 中英双语字幕（中文主，英文副）
- 关键命令和参数用等宽字体高亮显示
