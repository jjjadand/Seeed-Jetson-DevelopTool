# GUI 使用指南

## 安装

```bash
pip install PyQt5
pip install -e .
```

## 启动

```bash
seeed-jetson-flash gui
```

## 功能

### 1. 刷写固件选项卡
- 选择产品和 L4T 版本
- 配置刷写选项
- 查看进度和日志

### 2. Recovery 模式选项卡
- 查看进入 Recovery 模式的详细教程
- 包含步骤、图片链接

### 3. 关于选项卡
- 软件信息
- 相关链接

## 设计风格

采用 Seeed Studio 品牌色：
- 主色: #8DC21F (Seeed 绿)
- 辅色: #003A4A (Seeed 蓝)
- 渐变标题栏
- 圆角设计
